Como instalar

        
Seleciona o teu cliente de email para instruções específicas.

      

      

        
        
        
        
        
      


      

        
    
          
  1. Clica em Descarregar .htm para obter o ficheiro da assinatura.
    2. 
          
  2. Abre o Explorador de Ficheiros e cola este caminho na barra de endereço: %APPDATA%\Microsoft\Signatures\
    3. 
          
  3. Move o ficheiro .htm descarregado para essa pasta.
    4. 
          
  4. No Outlook, vai a Ficheiro → Opções → Correio → Assinaturas…
    5. 
          
  5. Seleciona a assinatura e define-a como padrão para novas mensagens e respostas.
    6. 
        

        
Porquê este método? Colar diretamente no editor de assinaturas do Outlook funciona na maioria dos casos, mas o Outlook limpa algum HTML ao colar. Substituir o ficheiro .htm preserva o código intacto.

    2. 
          
  2. No Outlook para Mac, vai a Outlook → Definições → Assinaturas.
    3. 
          
  3. Clica em + para criar uma nova assinatura e dá-lhe um nome.
    4. 
          
  4. Cola o conteúdo no editor (⌘V).
    5. 
          
  5. Associa a assinatura à tua conta em Escolher assinatura padrão.
    6. 
        

      


      

        
    
          
  1. Clica em Copiar renderizado acima.
    2. 
          
  2. Abre o Outlook Web (outlook.office.com) e clica no ícone de engrenagem.
    3. 
          
  3. Vai a Email → Redigir e responder.
    4. 
          
  4. Cola o conteúdo na caixa de assinatura.
    5. 
          
  5. Ativa as opções "incluir em novas mensagens" e "em respostas/reencaminhamentos".
    6.

    7. 
        

      


      

        
    
          
  1. Abre o Mail e vai a Mail → Definições → Assinaturas.
    2. 
          
  2. Cria uma nova assinatura para a tua conta (escreve qualquer texto temporário).
    3. 
          
  3. Fecha o Mail completamente (⌘Q).
    4. 
          
  4. No Finder, pressiona ⌘⇧G e cola: ~/Library/Mail/V10/MailData/Signatures/
    5. 
          
  5. Localiza o ficheiro .mailsignature mais recente e substitui o seu conteúdo pelo HTML (usa "Copiar HTML" acima).
    6. 
          
  6. Clica com o botão direito no ficheiro → Obter Informações → ativa "Bloqueado".
    7. 
          
  7. Reabre o Mail.
    8. 
        

        
Nota: O número da pasta V10 pode variar consoante a versão do macOS. Procura a pasta V* mais recente.

  2. Abre o Gmail e clica no ícone de engrenagem → Ver todas as definições.
    3. 
          
  3. No separador Geral, desce até Assinatura.
    4. 
          
  4. Clica em Criar nova, dá-lhe um nome e cola o conteúdo no editor.

Antes de adotar, testa

  
Envia um email para ti próprio e abre em diferentes clientes. Confirma cada ponto abaixo:

  

    
As imagens carregam sem ser necessário clicar em "mostrar imagens".

    
O alinhamento mantém-se igual em desktop e telemóvel.

    
Os números de telefone não aparecem a azul automático (iOS).

    
Os links do email e website funcionam corretamente.

    
Em Dark Mode, o logo e separador continuam visíveis.

    
Os ícones sociais abrem as páginas corretas da CSOL.

  

  

    Nota técnica. Nenhum HTML de assinatura renderiza perfeitamente em todos os clientes — o objetivo é "aceitável em todos, ótimo nos principais". Se usares Exclaimer, CodeTwo ou outro sistema corporativo, contacta a TI antes de instalar.
